FAST TRACK MOTORS — 7-PAGE WEBSITE
===================================

PAGES
  index.html     — Home
  services.html  — Services overview (6 services, detailed)
  mot.html       — MOT landing page (highest-search page) + FAQ
  pricing.html   — Transparent pricing tables
  about.html     — About / story / values / stats
  reviews.html   — Reviews + rating breakdown
  contact.html   — Booking form + contact + map

FOLDERS
  css/system.css — shared design system (colours, type, components)
  js/main.js     — shared interactions (nav, reveals, counters, FAQ)
  logo.jpg       — the brand logo

TO VIEW
  Open index.html in any browser. Everything is linked and works offline.

TO DEPLOY (Vercel)
  Drag this whole folder into Vercel, or push to GitHub and import.
  No build step needed — it's static HTML/CSS/JS.

DESIGN NOTES
  Fonts: Sora (display) + Inter (body) + JetBrains Mono (data), via Google Fonts
  Colours: defined as CSS variables in css/system.css — easy to retheme
  Fully responsive, mobile sticky call/book bar, reduced-motion respected
